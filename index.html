DIO Bank Pro

Premium Banking System — TypeScript, Clean Architecture & SOLID

⚡ TypeScript 🏗️ Clean Architecture 🧪 39 Tests 📦 Node.js ✅ CI/CD

🎯 Sobre

O DIO Bank Pro é uma evolução premium do desafio TypeScript da Digital Innovation One. Aplica Clean Architecture, princípios SOLID, OOP avançado e testes automatizados em um sistema bancário completo.

🏦 Funcionalidades

Conta Pessoal

Depósitos, saques e histórico de transações com validações robustas.

Conta Empresarial

Empréstimos empresariais até R$ 10.000 com controle de limites.

Conta Bônus

+10% de bônus em todo depósito. Recompensa automática.

Validações de Domínio

Erros tipados: saldo insuficiente, conta inativa, valor inválido.

🚀 Quick Start

git clone https://github.com/matheusflorindo32/dio-bank-pro.git
cd dio-bank-pro
npm install
npm run dev
# ou: npm test (39 testes automatizados)

🛠️ Tech Stack

TypeScript
Node.js
Vitest
ESLint
Prettier
GitHub Actions

🏛️ Arquitetura

4 camadas bem definidas: Domain (entidades), Application (casos de uso), Infrastructure (repositórios, loggers) e Presentation (CLI/Console). Cada camada depende apenas da camada interna — nunca do exterior.
