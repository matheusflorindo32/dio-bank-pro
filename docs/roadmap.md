# Roadmap

## Curto Prazo (v1.1.0)

- [ ] Persistência em SQLite
- [ ] CLI interativa completa com menus
- [ ] Transferência entre contas
- [ ] Taxa de manutenção mensal
- [ ] Extrato em formato PDF

## Médio Prazo (v1.2.0)

- [ ] API REST com Express/Fastify
- [ ] Autenticação JWT
- [ ] Docker e Docker Compose
- [ ] Swagger/OpenAPI documentation
- [ ] Frontend web em Vue/React

## Longo Prazo (v2.0.0)

- [ ] Microserviços
- [ ] Event sourcing para transações
- [ ] CQRS (Command Query Responsibility Segregation)
- [ ] Cache com Redis
- [ ] Filas com RabbitMQ/Apache Kafka
- [ ] Observabilidade (metrics, logs, traces)

- [ ] Mobile app
